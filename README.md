# Dashboard Admin Quote List App 

A Lists of all motivational quotes, riddle, poetry, motivational speech from all over the world built with React, Redux, tailwind (admin side). 

## Features

- Quote
- Author
- Tagging 
- Category
- CRUD

## Getting started

1. npm install
2. npm start

## Dependencies

 - redux
 - tailwind
 - react router dom
 - axios
